## iOS Keyword Field Strategy (100 characters max)

**EN (95 chars):** `anonymous,crush,secret,admirer,dating,mystery,voice,filter,blur,photo,reveal,heartbeat,shh,love,romance`

**FR (93 chars):** `anonyme,crush,secret,admirateur,rencontre,mystere,voix,filtre,flou,photo,reveal,battement,shh,amour`

Rules applied:
- No spaces (Apple strips them)
- No duplicate words from title/subtitle (Apple indexes those separately)
- Singular form only (Apple indexes both singular and plural)
- No competitor brand names (Apple rejects them)
- Comma-separated, no trailing comma

## ASO Best Practices Applied

### Title (30 chars max)
- Include primary keyword ("Anonymous Crush")
- Brand name first for recognition
- Dash separator for readability

### Subtitle (30 chars max)
- Hook with mystery and curiosity
- Second keyword opportunity ("secret")
- Speaks directly to the user ("you")

### Keywords (100 chars max)
- Mix of high-volume generic terms and long-tail niche terms
- No words repeated from title or subtitle
- All lowercase, comma-separated, no spaces

### Description
- First 3 lines are visible without "Read More" — make them count
- Features as bullet points for scannability
- Privacy emphasis (key differentiator in this category)
- Call-to-action at the end
- Natural keyword density without stuffing

### Screenshots
- Show the user journey: send → clue → voice → blur → reveal
- Captions with benefit-driven copy, not feature names
- First screenshot = strongest hook (most users only see 1-2)

### Localization
- Full FR translation (not just keywords)
- FR keywords researched separately (different search patterns)
- Cultural adaptation of copy tone (tutoiement in FR)

## Update Cadence

- **Monthly**: Review keyword rankings, adjust keyword field
- **Quarterly**: Refresh screenshots and description copy
- **Per release**: Update "What's New" with keyword-rich release notes
- **Seasonal**: Holiday-themed promotional text (Valentine's Day is peak)
